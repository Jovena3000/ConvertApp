📋 Caminhos para instalar no Android
✅
PWA via Chrome — Instala agora, sem compilar
Hospede a pasta 📱 PWA/ em HTTPS gratuito (Netlify Drop), abra no Chrome Android → menu ⋮ → Adicionar à tela inicial. Ícone, splash e offline funcionam.
⚡
TWA via Bubblewrap — APK real com 1 comando
Hospede a PWA em HTTPS, depois execute no terminal:
npm i -g @bubblewrap/cli && bubblewrap init --manifest <URL>/manifest.json && bubblewrap build
O arquivo app-release-signed.apk é gerado automaticamente.
🌐
PWABuilder — APK grátis sem instalar nada
Hospede a PWA em HTTPS → acesse pwabuilder.com → cole a URL → Package for Stores → Android → baixe o app-release.apk.
🔨
Android Studio — APK com controle total
Abra a pasta 📦 AndroidStudio/ → aguarde Gradle sync → Build → Generate Signed APK → escolha Debug (sem keystore) → instale o app-debug.apk.
❌
app-release.apk direto no ZIP — impossível
Um APK requer compilação com Android SDK (javac + d8 + aapt2 + zipalign + apksigner). Isso não é possível em um navegador. Use um dos métodos acima.
