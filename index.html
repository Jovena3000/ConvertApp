⚡ Como instalar no Android

Método 1 — PWA (imediato): Hospede a pasta 📱 PWA/ em HTTPS gratuito (Netlify/Vercel), abra no Chrome Android → menu → "Adicionar à tela inicial".

Método 2 — APK real: Abra 📦 AndroidStudio/ no Android Studio → Build → Generate Signed APK. Instale o .apk diretamente.

Método 3 — Sem Android Studio: Hospede a pasta PWA, acesse pwabuilder.com, cole a URL e baixe o APK assinado grátis.
