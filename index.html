⚡ 3 Formas de Instalar
✅ MAIS FÁCIL · SEM COMPILAR
Método 1 — PWA via Chrome (imediato)
1
Extraia a pasta 📱 PWA/ do ZIP
2
Arraste a pasta em netlify.com/drop — hospedagem HTTPS gratuita em 30 segundos
3
No celular Android: abra a URL no Chrome → menu ⋮ → "Adicionar à tela inicial"
4
✅ App instalado com ícone, splash e funciona offline!
📦 APK REAL · SEM ANDROID STUDIO
Método 2 — APK via PWABuilder (gratuito)
1
Hospede a pasta 📱 PWA/ no Netlify (método 1 passo 1 e 2)
2
Acesse pwabuilder.com, cole a URL hospedada e clique Start
3
Clique em Package for Stores → Android → Download Package
4
Baixe o app-release.apk assinado. Habilite "Fontes desconhecidas" e instale!
⚙️ APK NATIVO · ANDROID STUDIO
Método 3 — Compilar APK localmente
1
Extraia a pasta 📦 AndroidStudio/ do ZIP
2
Abra o Android Studio → File → Open → selecione a pasta
3
Aguarde o Gradle sync (~2 min na 1ª vez). Gradle wrapper já incluso no ZIP!
4
Menu Build → Build Bundle(s)/APK(s) → Build APK(s)
5
O arquivo app-debug.apk estará em app/build/outputs/apk/debug/
